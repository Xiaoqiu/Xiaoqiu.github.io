系统操作

  • 查看版本
    select version();

  • 链接本地数据库
    mysql -u用户名 -p密码

  • 链接远程数据库
    mysql -h110.110.110.110 -uroot -pabcd123
    （注:u与root可以不用加空格，其它也一样）

操作命令

show databases;
use 数据库名;
show tables;
describe 表名;
显示表格列的属性：show columns from tableName；

数据库操作命令

  • 创建数据库并指定编码
    注意：要在 Mysql 中保存 4 字节长度的 UTF-8 字符，需要使用 utf8mb4 字符集，但只有 5.5.3 版本以后的才支持

  • 将sql文件导入数据库
    mysql -u 用户名 -p 数据库名 < 数据库名.sql
    mysql -u abc -p abc < abc.sql
