redis

Posted on 2019-01-28 | In redis

安装redis

  • 1 redis在第三方的源里，首先添加源，最常用的是epel源
    yum install epel-release
  • 2 然后就可以安装redis了
    yum install redis
  • 3 安装完成以后配置文件/etc/redis.conf ， 启动redis服务端
    redis-server /etc/redis.conf
    systemctl start redis
    设置开机启动
    systemctl enable redis
  • 4 查看是否启动
    ps -ef|grep “redis”
  • 5 启动客户端
    redis-cli

Replication 主从复制

Redis的replication机制允许slave从master那里通过网络传输拷贝到完整的数据备份，从而达到主从机制。为了实现主从复制，我们准备三个redis服务，依次命名为master()，slave1，slave2。

配置主服务器

  • 必须配置/etc/redis.conf的bind参数，配合为127.0.0.1 和本机的地址192.168.33.10, 因为redis-slave的从服务器需要通过本机地址192.168.33.10访问redis-master.

配置从服务器

  • 配置两台从服务器redis.conf文件的slaveof参数，注释掉bind参数

  • 从redis2.6版本开始，slave支持只读模式，而且是默认的。可以通过配置项slave-read-only来进行配置。
